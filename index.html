Start keeping good coding habits from A+B Problem

Share

Now, we believe every OIer can write a code of A+B problem very easily.

So easy,isn’t it? But if I say there are many problems in this code, can you believe?

First,do you truly end this program? Where is your return 0?

Second,do you create a variable to store the answer?

Maybe you think it is not necessary. But it is. As you see, a and b are both integers. If the value of a+b is greater than 2147483647,or less than -2147483648,the output will be wrong. So we can create a variable to store the answer.

Third,do you think the input and the output is too slow?If so,we can change our way to input or output.
